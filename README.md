# nollae Blog

[👉 nollae 블로그 바로가기](https://nollae.github.io/)

`Jekyll`을 활용해 블로그를 만들었고, 블로그 템플릿을 기반으로 특정 요소 및 기능들을 커스터마이징하여 원하는 방향성에 맞게 블로그를 구현

- 블로그 템플릿
  - [chirpy github](https://github.com/cotes2020/jekyll-theme-chirpy)
  - [chirpy sample](https://chirpy.cotes.page/)
 
- 커스터마이징 부분
  - lignt mode일때, code를 입력했을 때 dark mode css로 적용되도록 수정.
  - code 입력 시, word 색상 수정.
  - iframe을 활용해 code result 값이 블로그 css에 영향주지 않고 실행되도록 구현.
  - 블로그 전체적인 색상 및 구성 수정.
